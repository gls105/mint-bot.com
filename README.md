# mint-bot.com

Personal site of Martin Abondano (mvrtn) — vibe coder, Miami.

One HTML file, zero frameworks, zero build steps. Featuring a math-generated
braille duck that walks down the page leaving footprints, and a bread-throwing
mini-game.

**Live:** [mint-bot.com](https://mint-bot.com) · [mint-bot.pages.dev](https://mint-bot.pages.dev)

## Deploy

```bash
cp index.html dist/ && npx wrangler pages deploy dist --project-name=mint-bot
```
